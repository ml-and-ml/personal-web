# Personal Homepage Requirements

## Overview
A single-page personal homepage with a dark theme, featuring smooth navigation and interactive elements.

## Sections

### 1. Header Navigation
- Fixed header at top of page
- Navigation links to all sections
- Smooth scroll behavior

### 2. Hero Section
- Visually unique and interactive design
- Animated/dynamic elements
- Eye-catching introduction

### 3. Publications Section
- Two subsections:
  - **Peer Reviewed Publications**
  - **Abstracts**
- Display first 3 items in each subsection by default
- "Show More" button that expands full list to the right side of the page
- Clean, readable layout

### 4. Blog Section
- Blog-style articles display
- Article cards with title, date, excerpt
- Expandable or link to full content

### 5. About Me Section
- Personal bio
- Background information
- Contact details or links

## Design Requirements
- **Dark theme** throughout
- **Teal/Cyan accent colors** (no purple)
- Modern, clean aesthetic
- Responsive design
- Smooth animations and transitions
- Interactive hover states

## Technical Stack
- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript (no frameworks)

## Interactivity
- Smooth scroll navigation
- Publication list expansion (slides in from right)
- Hero section animations
- Hover effects on interactive elements
